01 // bio.exe

Built like a terminal. Pushed like a confession.

HTG runs as a shared release system for two voices. Stretty carries the fractured, depressive pull. Abraxas drives the sharper cut, ritual energy, and hostile edge. Together it lands somewhere between underground rap, emo collapse, and industrial static.

boot_01

Signal Formation

Started as isolated demos, rough notes, and private drops. The sound leaned cold, damaged, and stripped of filler from day one.

boot_02

Dual Process

Stretty and Abraxas merged the project into one release identity. Two artists, one shell, one tag, one visual language.

boot_03

HTG Runtime

Now the project runs as a full release page for singles, visuals, tour updates, merch drops, and whatever gets dragged out next.
